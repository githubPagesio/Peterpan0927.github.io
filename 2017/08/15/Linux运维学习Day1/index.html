Linux运维学习Day1

Linux命令的格式：

  command [-option] [parameters] ...
  e.g->mike@mike:~$ ls -a /home/mike

ls命令(list的简写)：列出当前目录下的文件和文件夹

e.g：
pzp@ubuntu:/home$ ls
cpp  python
pzp@ubuntu:/home$ ls -l
drwxr-xr-x 2 cpp YYY 4096 10月 13 09:58 cpp
-rw-rw-r 1 python python 18 10月 12 18:47 dump.rdb 
pzp@ubuntu:/home$ ls -a
. .. cpp  python

1.ls -l是列出详细的信息，如果前面是以’-‘开头的就是文件，如果是’d’开头则是文件夹。

2.ls -a则是列出所有的文件包括隐藏的文件和文件夹。

3.ls [路径]是列出选择的路径下的文件和文件夹。

4.ls -l命令所列出的信息中时间前的那一串数字表示的其所占的内存大小是多少k。

pwd命令

返回当前文件夹的路径

cd命令

切换目录

e.g:
cd ..  #返回上一目录
cd ./xxx/ #进入本目录下的xxx文件夹
cd /xxx/xxx #使用绝对路径
cd xxx #使用相对路径
cd - #进入上次所在的目录
cd ~ #切换回主目录

clear命令

清空屏幕，但其实之前的命令还有保留，上滑就能看到。

tab命令

当文件或者文件夹名字过长的时候，可以使用tab键来进行补全。

touch命令

创建一个文件。

more命令

当文件中的信息很多时，可以采用more来分屏显示,通过空格键往下：more text.txt

cat命令

显示出该文件的所有内容：cat text.txt

合并两个文件为一个文件：cat test1.txt test2.txt > test3.txt(先将两个文件合并再重定向到一个新的文件)

创建和删除文件夹

mkdir  xxx  #创建文件夹

rmdir  xxx #只允许删除空文件夹

rm xxx  #删除文件

rm xxx -r #递归删除，删除文件夹必加'-r'参数

rm xxx -f #强制删除

命令查询

当你不清楚某个命令的用法的时候，可以通过查询命令来了解：

man ls
ls --help

其中的man是manual的缩写，意为手册，其中第一节的手册是对标准命令的查询，一共有八节手册.

通配标识符

通过通配标识符可以对文件进行快速查找：

e.g->ls xxx*.x意为对名称中含有xxx,后缀为.x的文件开始快速查询。

重定向

Linux允许将命令执行结果重定向到一个文件，本应显示在终端上的内容保存到指定文件中。e.g:

ls > text.txt #test.txt如果不存在，则创建，存在则覆盖其内容
ls >> text.txt #test.txt不存在同上，反之在保留原始内容同时加上新内容

管道

管道：一个命令输出可以作为另一个命令的输入。

软链接和硬链接

软链接： ln -s 源文件 链接名 #删除原文件之后失效

硬链接： ln 源文件 链接名  #相当于给该文件多加了一个名字，删除原文件之后链接依旧有效

在linux中一个文件可以有多个文件名，将文件删除的时候如果该文件有多个名字，首先删除的是这个名字，当最后一个名字被删除后，才会删除文件。

文本搜索命令

grep  '关键词'  文本  
grep -n '关键词' 文本  #搜索之后分行显示
grep -n '^关键词' 文本 #搜索以关键词为开头分行显示
grep -n '关键词$' 文本 #搜索以关键词为结尾分行显示
grep '[Ss]igna[Ll]' 文本 #搜索匹配【】里中一系列字符中的一个

文件搜索命令

可以配合通配标识符使用，搜索会更加便捷。

拷贝文件和移动文件

cp 文件或文件夹  文件夹 #前者是被拷贝方，后者是拷贝方

mv 文件或文件夹  文件夹 #同上
参数表
-i 交互式拷贝
-r 递归式拷贝
-f 强制拷贝
-v 显示拷贝进度

重命名

mv 文件名 文件名

压缩和解压缩（.gz格式）

tar -cvf  xxx.tar 压缩文件1 压缩文件2..... #打包
gzip xxx.tar  #压缩
gzip -d  xxx.tar.gz  #解压缩
tar -xvf xxx.tar #解包
tar -zcvf xxx.tar.gz 压缩文件1 压缩文件2..... #一键压缩
tar -zxvf xxx.tar.gz -C 路径 #将文件一键解压到指定目录下

如果是.bz2格式的，只需要把参数里面的’z’改成’j’即可，其余不变。

如果是zip格式的：

zip -r 目标文件（没有扩展名） 源文件 #压缩文件

unzip -d 解压缩后目录（如果没有则创建） 压缩文件  #解压文件

搜索命令

which ls

与系统相关的命令

1.查看当前日历：cal (月份)(年份)

2.显示时间或者设置时间：date MMDDhhmmCCYY.ss (CC为年的前两位，yy为后两位，MM为月，mm为分钟)

3.查看进程的信息：ps ，参数表参考ps命令总结，如果想结束一个无限循环的程序，可以另开一个窗口，用ps -aux找到这个进程的进程号（PID），就可以用kill 进程号来结束这个进程，如果有些进程无法被杀死，可以用kill -9 进程号。

4.动态显示进程：top 参数表参考top命令详解

5.结束进程：kill 进程号，必要时可以使用kill -9 进程号

6.重启，关闭电脑：reboot(重启操作系统) shutdown参数表 init 0(关机) init 6(重启)

7.检测磁盘空间的占用情况和空余情况：df

8.检测当前目录所占磁盘空间：du -a

9.查看或配置网卡信息：ifconfig
